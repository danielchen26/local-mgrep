Quick start

Install, index, search.

The core workflow needs Python 3.9+, Ollama, and one local embedding model. No hosted account, API key, organization setup, or remote code upload is required. 

pip install local-mgrep
ollama pull mxbai-embed-large

mgrep index /path/to/repo --reset
mgrep search "how does auth work?" -m 10 --json

Capabilities

Built for private repos and coding-agent context.

🔎

Semantic search

Search by intent instead of exact keywords, using local Ollama embeddings.

🧠

Hybrid ranking

Blend vector similarity with exact lexical code-term matches for practical ranking.

🧩

Diversified top-k

Cap repeated chunks from the same file so more relevant files fit in context.

📦

Agent-ready JSON

Return path, line range, language, score, and snippet for scripts and agents.

🛡️

Local privacy boundary

Keep source scanning, vectors, retrieval, and optional answers on your workstation.

⚡

Watch mode

Keep the index warm while editing with incremental add, update, and delete handling.

Architecture

A small local pipeline you can inspect.

Files flow through ignore rules, tree-sitter-aware chunking, Ollama embeddings, SQLite storage, local ranking, and source-cited output.
